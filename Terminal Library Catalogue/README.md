# Terminal Library Catalogue

<p>
  This repository contains a C++ library catalogue application developed as a final project for Computer Programming 1. The program operates entirely in the terminal, providing an interactive menu where users can browse through various book categories. Instead of creating new book entries from scratch, the system functions similarly to a shopping cart where users select existing books from predefined catalogues and add them to a personal favorites collection.  

Key Features:

  Categorized Browsing: Users can navigate through five distinct book genres: Fiction, Science Fiction, Mystery, Fantasy, and Computer Science.  

  Favorites System: Books can be selected from the catalogue and saved to a personal favorites list, managed under the hood using C++ vectors.  

  Duplicate Prevention: The program automatically checks if a book is already in the favorites list to prevent duplicate entries before adding it.  

  List Management: Users have the ability to view their saved books, cancel additions, or completely clear their favorites list.  

  Live Clock: Displays the current date and time on the terminal menus.  

Technical Details:

  Built using object-oriented principles with a central Book class to structure book attributes like ID, title, author, and category.  

  Utilizes a do-while loop and switch statements for continuous, interactive main menu navigation.
</p>
